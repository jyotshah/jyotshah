WorkNest is a mobile application designed for restaurant owners and small business managers to easily organize and oversee their operations. DJ-AJAX ‚Äì Dynamic Django Web Application

DJ-AJAX is a full-featured web application built with Django and enhanced using AJAX for a smooth, real-time user experience. Hosted on PythonAnywhere, it simulates a lightweight social platform where users can manage posts, profiles, and interactions.

The application starts with a secure user authentication system (which is commented out right now for easy access :) ). Featuring clean navigation, profile management, media handling, and real-time interactions, DJ-AJAX demonstrates strong full-stack development skills with an emphasis on seamless user experience.
